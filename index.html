5th Semester Syllabus

54H

Theory of Computation

Unit 1 (12 Hours)

Introduction to Compiler Design and Lexical Analysis, Overview of Compilation and its Phases, Lexical Analysis: Role of a Lexical Analyzer, Finite Automata: NFA & DFA, Regular Expressions, Lexical Grammar, Tools: Lex & Flex.

Unit 2 (12 Hours)

Syntax Analysis: Role of a Parser, Context-Free Grammars, Parse Trees, Ambiguity, Top-Down Parsing (Recursive Descent, LL(1)), Bottom-Up Parsing (Shift-Reduce, SLR, LR(1), LALR), Error Handling.

Unit 3 (10 Hours)

Syntax-Directed Translation & Semantic Analysis, SDD, Attributes, Dependency Graphs, Attribute Grammars, Applications of Syntax-Directed Translation.

Unit 4 (10 Hours)

Intermediate Code Generation & Symbol Table: Structures, Scope, Three Address Code, Quadruples, Triples, Syntax Trees, DAGs, Type Checking, Conversions, Control Flow, Intermediate Code for Procedures & Run-Time Environments.

Unit 5

Code Generation & Optimization: Basic Blocks, Flow Graphs, Register Allocation, Peephole Optimization, Removal of Redundant/Unreachable Code, Machine-Dependent Optimization.

56H

Data Mining and Data Warehousing (DMDW)

Unit 1 (12 Hours)

Introduction to Data Mining, Technologies, Issues, Data Objects, Attributes, Statistical Descriptions, Data Preprocessing (Integration, Cleaning, Transformation, Reduction, Discretization).

Unit 2 (12 Hours)

Introduction to Data Warehousing, Need, Concepts, Architecture, Models, ETL, Metadata, Trends, OLAP vs OLTP, Data Cube, Schemas, Warehouse Design & Usage.

Unit 3 (12 Hours)

Mining Frequent Patterns: Associations, Correlations, Apriori, FP-Growth, Classification (Decision Tree, Bayes, Rule-based, KNN, Logistic Regression, SVM).

Unit 4 (12 Hours)

Cluster Analysis: K-means, K-medoids, Hierarchical, DBSCAN. Graph & Network Data, Anomaly Detection, Outlier Analysis, Text Mining (Preprocessing, TF-IDF, Classification, Clustering), Web Mining (Content, Structure, Usage).

Unit 5 (8 Hours)

Challenges of Outlier Detection, Supervised & Semi-supervised, Unsupervised, Statistical & Proximity-based methods, Data Mining Approaches.

56H

Cloud Computing

Unit 1 (12 Hours)

Cloud Computing: Introduction, Types of cloud: private, public, hybrid. Hardware & Infrastructure: IaaS, PaaS, SaaS. Benefits & Challenges, Virtualization (CPU, Memory, I/O Devices), Virtual Clusters, Data-center Automation.

Unit 2 (14 Hours)

Cloud Infrastructure: Architectural Design, Layered Cloud Architecture, Inter-Cloud Resource Management, Provisioning, Deployment. Cloud Applications: Web Services Deployment, SLA Management.

Unit 3 (10 Hours)

Cloud Economics: Choosing Platforms (Amazon, Microsoft, Google, etc.), CloudSim Benefits & Architecture, Scheduling Algorithms.

Unit 4 (10 Hours)

Cloud Security: Challenges, SaaS Security, Risk Management, Monitoring, Security Architecture, Data & Application Security, VM Security, Identity & Access Management.

Unit 5 (10 Hours)

Cloud Service Development & Trends: Cloud-based Applications, Platforms, Tools, Deployment, Edge Computing, Serverless, Containerization, AI in Cloud, Quantum Cloud.

48H

Advanced Java

Unit 1 (12 Hours)

Basics of JDBC: Need, Drivers, Architecture, Components. Programs to connect DB, Execute Queries, ResultSet, Metadata, Callable Statements, Stored Procedures.

Unit 2 (12 Hours)

Web Architecture: HTML Basics, Tags, Layout, Forms. Servlets: Advantages, Architecture, HTTP Protocol, Types of Servlets, Life Cycle, doGet/doPost, HTML Output, DBMS Communication, Servlet Chaining, Redirect, Filters, Listeners, Session Tracking.

Unit 3 (10 Hours)

Java Server Pages: Introduction, Anatomy, Execution, JSP Engine, Objects, Directives, Elements, Beans, Actions (useBean, setProperty, getProperty).

Unit 4 (14 Hours)

Hibernate: Introduction, Architecture, O/R Mapping, Configuration, Development, HQL.

Unit 5

Spring & SpringBoot: Spring Core, IoC, MVC, Hibernate Integration, SpringBoot Basics, Dependency Injection, JPA, WebApp MVC.

50H

Computer Networks (DCCN)

Unit 1 (10 Hours)

Introduction to Computer Networks, Components, Devices, Topology, OSI model, layers in OSI model, TCP/IP Model, Address types (Physical, Logical, Port, Application-Specific Addresses), Analog and Digital Signal. Transmission Impairment (Attenuation, Distortion, Noise), Data Rate Limits (Nyquist Bit rate, Shannon Capacity). Performance (Bandwidth, Throughput, Latency, Jitter). Transmission Media (Guided Media, Unguided Media).

Unit 2 (10 Hours)

Physical Layer: Line encoding, Block coding, Binary data to Digital Signal Conversion, Digital to Analog Signal Conversion, Analog to Analog Signal Conversion, Analogue to Digital Signal conversion. Digital Transmission types (Serial and Parallel), Transmission Modes (Simplex, Half-Duplex and Full-Duplex), Multiplexing (FDM, TDM, WDM, CDM). Switching and Types: Circuit Switched Networks, Datagram Networks, Virtual Circuit Networks.

Unit 3 (10 Hours)

Data link layer: Introduction, Error Detection and Correction techniques (Checksum, Cyclic Redundancy Check, Hamming Code (1-bit error rectification)), Framing, HDLC Frame Structure and Types, HDLC (NRM, ABM, ARM), Point to Point Protocol, Flow and Error Control, Noiseless Channels protocols (Simplest, Stop–and-Wait), Noisy Channels protocols (Stop-and-Wait ARQ, Go-Back-N ARQ, Selective Repeat ARQ).

Unit 4 (10 Hours)

Medium Access sub layer: Multiple Access - Random Access Protocols (ALOHA, CSMA (non-persistent, 1-persistent, p-persistent), CSMA/CD, CSMA/CA), Controlled Access Protocols (Reservation, Polling, Token Passing), Channelization techniques (FDMA, TDMA, CDMA). Network Layer: Inter-networking, IP Address, IPv4 (Header structure, Class types, Sub-netting), IPv6 (Header structure, Address structure), IPV4 to IPV6 Address Mapping, Address Resolution Protocol, ICMP.

Unit 5 (10 Hours)

Transport Layer: User Datagram Protocol, Transmission Control Protocol, Process-To-Process Delivery, Congestion Control and its Type, Quality of Service. Application Layer: Domain Name System - Address Resolution, TELNET, E-MAIL (SMTP, POP3), FTP, HTTP.

36H

Engineering Economics and Costing

Unit 1 (10 Hours)

Fundamentals of Microeconomics for Engineering Applications: Meaning, Nature, Scope of Engineering Economics, Law of Demand and its exceptions, Determinants of demand, Elasticity of demand & its measurement (Simple numerical problems to be solved), Law of supply and its exception, Determination of market equilibrium (Simple numerical problems to be solved). Law of variable proportion, Law of returns to scale.

Unit 2 (6 Hours)

Market Structures and Cost Analysis: Cost and revenue function, Price and output Determination under perfect competition (Simple numerical problems to be solved), Break Even Analysis - Linear approach (Simple numerical problems to be solved).

Unit 3 (8 Hours)

Time Value of Money and Investment Analysis: Time value of money – Application of interest formulas, Cash flow diagrams, Present worth method, Future worth method, Annual worth method, Internal rate of return method.

Unit 4 (6 Hours)

Project Evaluation and Depreciation: Cost benefit analysis for public projects, Depreciation - Depreciation of capital assets, causes of depreciation, Methods of calculating depreciation (Straight line method, Declining balance method).

Unit 5 (6 Hours)

Overview of Banking, Inflation, and National Income: Functions of commercial bank, Functions of Central Bank. Inflation - Meaning of inflation, types, causes, measures to control inflation. National Income - Concepts of national income & Method of measuring national income.
