5th Semester Syllabus

Theory of Computation

Unit 1 (12 Hours)

Introduction to Compiler Design and Lexical Analysis, Overview of Compilation and its Phases, Lexical Analysis: Role of a Lexical Analyzer, Finite Automata: NFA & DFA, Regular Expressions, Lexical Grammar, Tools: Lex & Flex.

Unit 2 (12 Hours)

Syntax Analysis: Role of a Parser, Context-Free Grammars, Parse Trees, Ambiguity, Top-Down Parsing (Recursive Descent, LL(1)), Bottom-Up Parsing (Shift-Reduce, SLR, LR(1), LALR), Error Handling.

Unit 3 (10 Hours)

Syntax-Directed Translation & Semantic Analysis, SDD, Attributes, Dependency Graphs, Attribute Grammars, Applications of Syntax-Directed Translation.

Unit 4 (10 Hours)

Intermediate Code Generation & Symbol Table: Structures, Scope, Three Address Code, Quadruples, Triples, Syntax Trees, DAGs, Type Checking, Conversions, Control Flow, Intermediate Code for Procedures & Run-Time Environments.

Unit 5

Code Generation & Optimization: Basic Blocks, Flow Graphs, Register Allocation, Peephole Optimization, Removal of Redundant/Unreachable Code, Machine-Dependent Optimization.

Data Mining and Data Warehousing (DMDW)

Unit 1 (12 Hours)

Introduction to Data Mining, Technologies, Issues, Data Objects, Attributes, Statistical Descriptions, Data Preprocessing (Integration, Cleaning, Transformation, Reduction, Discretization).

Unit 2 (12 Hours)

Introduction to Data Warehousing, Need, Concepts, Architecture, Models, ETL, Metadata, Trends, OLAP vs OLTP, Data Cube, Schemas, Warehouse Design & Usage.

Unit 3 (12 Hours)

Mining Frequent Patterns: Associations, Correlations, Apriori, FP-Growth, Classification (Decision Tree, Bayes, Rule-based, KNN, Logistic Regression, SVM).

Unit 4 (12 Hours)

Cluster Analysis: K-means, K-medoids, Hierarchical, DBSCAN. Graph & Network Data, Anomaly Detection, Outlier Analysis, Text Mining (Preprocessing, TF-IDF, Classification, Clustering), Web Mining (Content, Structure, Usage).

Unit 5 (8 Hours)

Challenges of Outlier Detection, Supervised & Semi-supervised, Unsupervised, Statistical & Proximity-based methods, Data Mining Approaches.

Cloud Computing

Unit 1 (12 Hours)

Cloud Computing: Introduction, Types of cloud: private, public, hybrid. Hardware & Infrastructure: IaaS, PaaS, SaaS. Benefits & Challenges, Virtualization (CPU, Memory, I/O Devices), Virtual Clusters, Data-center Automation.

Unit 2 (14 Hours)

Cloud Infrastructure: Architectural Design, Layered Cloud Architecture, Inter-Cloud Resource Management, Provisioning, Deployment. Cloud Applications: Web Services Deployment, SLA Management.

Unit 3 (10 Hours)

Cloud Economics: Choosing Platforms (Amazon, Microsoft, Google, etc.), CloudSim Benefits & Architecture, Scheduling Algorithms.

Unit 4 (10 Hours)

Cloud Security: Challenges, SaaS Security, Risk Management, Monitoring, Security Architecture, Data & Application Security, VM Security, Identity & Access Management.

Unit 5 (10 Hours)

Cloud Service Development & Trends: Cloud-based Applications, Platforms, Tools, Deployment, Edge Computing, Serverless, Containerization, AI in Cloud, Quantum Cloud.

Advanced Java

Unit 1 (12 Hours)

Basics of JDBC: Need, Drivers, Architecture, Components. Programs to connect DB, Execute Queries, ResultSet, Metadata, Callable Statements, Stored Procedures.

Unit 2 (12 Hours)

Web Architecture: HTML Basics, Tags, Layout, Forms. Servlets: Advantages, Architecture, HTTP Protocol, Types of Servlets, Life Cycle, doGet/doPost, HTML Output, DBMS Communication, Servlet Chaining, Redirect, Filters, Listeners, Session Tracking.

Unit 3 (10 Hours)

Java Server Pages: Introduction, Anatomy, Execution, JSP Engine, Objects, Directives, Elements, Beans, Actions (useBean, setProperty, getProperty).

Unit 4 (14 Hours)

Hibernate: Introduction, Architecture, O/R Mapping, Configuration, Development, HQL.

Unit 5

Spring & SpringBoot: Spring Core, IoC, MVC, Hibernate Integration, SpringBoot Basics, Dependency Injection, JPA, WebApp MVC.
